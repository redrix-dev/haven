## Updated Features

- Root-hosted web client + installable PWA baseline for `haven.redrixx.com`
- Near-instant web push path with wakeup dispatch and cron backstop
- Cleaner notification UX with a single `Enable/Disable Push Notifications` device toggle
- Auto service worker registration on supported secure contexts
- Voice settings stability fix for device select rendering on browsers that expose blank audio device IDs
- Expanded release validation workflow with scriptable signoff artifacts
